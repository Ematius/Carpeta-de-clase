# Para array

.sort() (modifica la array original) (Ingles = Clasificar)
.toSorted (Crea una array nueva)
Key Differences:
Return value: sort() doesn't explicitly return a value (it modifies the original array), while toSorted() returns a new, sorted array.

.filter() (Crea una nueva array)(Ingles= filtrar) crea un nuevo arreglo y agrega a él retornó true.

.split() (Crea un nuevo array) (Ingles = Dividir)

.toSpliced() (Crea un nuevo array) (Ingles = dividida)
